# User Preferences

## Level
- Full-stack developer — comfortable with general programming concepts
- Beginner with LangChain and LangGraph — do not assume prior knowledge of these frameworks
- Likes to understand what he is doing, not just copy-paste code

## Stack
- Python projects: uses `uv` for package management
- JS/TS projects: uses `pnpm`

## Expected behavior
- After each code change, ask: "Do you want me to explain this change?"
- If yes, explain simply: what the code does, why this choice, and how it fits into the rest
- Do not overload with unsolicited explanations
